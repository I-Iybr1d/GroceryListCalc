# GroceryListCalc
A grocery lister and price calculator using ionic framework

Current features:
- List Products
- Set default value price, quantity and discount for each product
- Calculates total list price

To Do:

# Instalation
1 - Install Ionic
- npm install -g cordova ionic
2 - Install Node Modules
- npm install
3 - Run Ionic server
- ionic serve
